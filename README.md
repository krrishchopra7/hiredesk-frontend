# 💼 HireDesk — Frontend

> Interview Pipeline Management System — React JS Dashboard

## 🚀 Tech Stack
- React JS 18
- Redux Toolkit (state management)
- React Beautiful DnD (drag and drop)
- Recharts (analytics charts)
- Axios (API calls)
- React Router DOM
- React Hook Form

## ✨ Features
- JWT based login with role based routing
- Dashboard with real time stats
- Drag and drop Kanban pipeline board
- Create job openings with interview rounds
- Add candidates with source tracking
- Analytics dashboard — funnel chart, pie chart, interviewer table
- Responsive design
